Model Analysis
01

What it does

Fits a linear equation y = β₀ + β₁x₁ + β₂x₂... to minimise Mean Squared Error between predicted and actual log(charges).

02

Strengths observed

Simple, interpretable, and established a strong baseline. Test R² of 0.8025 outperforms similar studies (e.g., Kaggle baseline R²=0.6951). Log transformation significantly aided performance.

03

Limitations

Cannot capture nonlinear relationships inherent in insurance data. Train R² (0.7295) lower than test (0.8025) suggests mild underfitting on complex patterns.

Why Gradient Boost Wins
01

Sequential correction

Each weak learner (tree) corrects residual errors of the previous, building a progressively stronger model.

02

Built-in regularisation

Gradient Boosting includes regularisation parameters that naturally prevent overfitting without manual tuning.

03

Consistent generalisation

Test MSE (0.0238) ≈ Train MSE (0.0239) — only 0.0001 gap, indicating excellent generalisation.

1. Data Loading & Cleaning
import pandas as pd import numpy as np from sklearn.preprocessing import LabelEncoder # Load dataset from Kaggle medical = pd.read_csv('insurance.csv') # Remove duplicate records medical = medical.drop_duplicates() print("Remaining records:", len(medical)) # Verify no missing values exist print(medical.isnull().sum()) # All zeros
2. Encoding & Feature Engineering
# Label encode categorical variables le_sex = LabelEncoder() le_smoker = LabelEncoder() medical['sex'] = le_sex.fit_transform(medical['sex']) medical['smoker'] = le_smoker.fit_transform(medical['smoker']) # Log10 transform target to reduce skew medical['charges'] = np.log10(medical['charges']) # Feature selection X = medical[['age', 'sex', 'bmi', 'smoker']] Y = medical['charges']
3. Train/Test Split
from sklearn.model_selection import train_test_split # 80% training, 20% testing — random_state=42 for reproducibility X_train, X_test, Y_train, Y_test = train_test_split( X, Y, test_size=0.2, random_state=42 ) print(f"Training samples: {len(X_train)}") # ~1070 print(f"Testing samples: {len(X_test)}") # ~268
Linear Regression
from sklearn.linear_model import \ LinearRegression # Instantiate and train lr_model = LinearRegression() lr_model.fit(X_train, Y_train) # Predict lr_test_pred = lr_model.predict(X_test) lr_train_pred = lr_model.predict(X_train) # Coefficients print(lr_model.coef_)
Decision Tree
from sklearn.tree import \ DecisionTreeRegressor # Tuned hyperparameters dt_model = DecisionTreeRegressor( max_depth=30, min_samples_split=50, min_samples_leaf=10, random_state=42 ) dt_model.fit(X_train, Y_train) dt_test_pred = dt_model.predict(X_test) dt_train_pred = dt_model.predict(X_train)
Gradient Boost
from sklearn.ensemble import \ GradientBoostingRegressor # Default params — no tuning needed gb_model = GradientBoostingRegressor( random_state=42 ) gb_model.fit(X_train, Y_train) gb_test_pred = gb_model.predict(X_test) gb_train_pred = gb_model.predict(X_train)
from sklearn.metrics import r2_score, mean_squared_error def evaluate_model(name, train_pred, test_pred, Y_train, Y_test): train_r2 = r2_score(Y_train, train_pred) test_r2 = r2_score(Y_test, test_pred) train_mse = mean_squared_error(Y_train, train_pred) test_mse = mean_squared_error(Y_test, test_pred) print(f"\n{name}") print(f" Train R²: {train_r2:.4f} | Test R²: {test_r2:.4f}") print(f" Train MSE: {train_mse:.4f} | Test MSE: {test_mse:.4f}") return {'train_r2': train_r2, 'test_r2': test_r2, 'train_mse': train_mse, 'test_mse': test_mse} # Evaluate all three models lr_metrics = evaluate_model("Linear Regression", lr_train_pred, lr_test_pred, Y_train, Y_test) dt_metrics = evaluate_model("Decision Tree", dt_train_pred, dt_test_pred, Y_train, Y_test) gb_metrics = evaluate_model("Gradient Boost", gb_train_pred, gb_test_pred, Y_train, Y_test)
